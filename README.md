# Analyser-un-systeme-vulnerable-pour-une-petite-entreprise
Projet réalisé lors de la certification Google Cybersecurity, mettant en pratique les connaissances en évaluation de vulnérabilité
# Contexte
Nous venons d'être embauché comme analyste en cybersécurité dans une entreprise de commerce électronique. L'entreprise stocke les informations sur un serveur distant de base de données, étant donné que de nombreux employés travaillent à distance dans le monde entier. Les employés de l'entreprise interrogent régulièrement le serveur pour trouver des clients potentiels. La base de données est ouverte au public depuis le lancement de l'entreprise il y a trois ans. En tant que professionnel de la cybersécurité, Nous reconnaissons que le fait de laisser le serveur de la base de données ouvert au public constitue une grave vulnérabilité.

Nous sommes chargé de réaliser une évaluation des vulnérabilités de la situation afin de communiquer les risques potentiels aux décideurs de l'entreprise. Vous devez rédiger un rapport expliquant en quoi le serveur vulnérable représente un risque pour les activités de l'entreprise et comment il peut être sécurisé.
